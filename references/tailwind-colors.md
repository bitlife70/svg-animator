# Tailwind CSS Color Palette (v3/v4)

SVG 애니메이션에 세련미를 더하기 위해 다음 색상 조합을 권장합니다.

## Dark Backgrounds (Slate / Zinc)
- Slate-900: `#0f172a` (Main background)
- Slate-800: `#1e293b` (Containers)
- Slate-400: `#94a3b8` (Secondary text / paths)

## Primary Accents (Blue / Sky)
- Sky-400: `#38bdf8` (Active data flow, glowing effects)
- Blue-500: `#3b82f6` (Primary nodes)

## Success & Info (Emerald / Cyan)
- Emerald-400: `#34d399` (Success signals)
- Cyan-400: `#22d3ee` (Information packets)

## Warning & Error (Amber / Rose)
- Amber-400: `#fbbf24` (Alerts)
- Rose-500: `#f43f5e` (Errors)

## Gradients Example
- Primary Flow: `Sky-400` to `Blue-600`
- Success Flow: `Emerald-400` to `Teal-500`
